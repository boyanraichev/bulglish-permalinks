== Description ==

This plugin converts Bulgarian cyrillic characters in slugs to Latin characters, according to the official rules for transliteration.

Cyrillic letters in URLs, while supported, are ugly and when copy/pasted around the internet are often encoded (converted to those ugly URLs looking like %DU^ED%â€¦)

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
